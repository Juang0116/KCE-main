# KCE Founder Response Lanes — 94

## Objetivo
Hacer visible una lectura operativa simple para el fundador:
- qué responder hoy,
- en qué ventana,
- y qué cola abrir primero.

## Ventanas recomendadas
- Mismo día: deals en proposal / checkout con señal de cierre.
- ≤12h: planes personalizados, contacto premium, handoff comercial.
- ≤2h: tickets post-compra y soporte con riesgo de experiencia.

## Cadena correcta
1. Chat / tours / plan personalizado / contacto.
2. Ticket si hay continuidad humana.
3. Deal si hay intención comercial real.
4. Task con ventana de respuesta.
5. Operador revisa leads + deals + tasks + tickets como un solo sistema.
