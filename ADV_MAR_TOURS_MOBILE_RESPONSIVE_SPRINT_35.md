# ADV MAR — Tours Mobile Responsive Sprint 35

## Focus
Refinar la experiencia mobile de `/tours/[slug]` para que la reserva no se sienta escondida ni invadida por demasiados elementos flotantes.

## Cambios principales
- CTA sticky mobile rediseñado como pill flotante más limpia
- eliminación del CTA WhatsApp dentro del sticky para reducir ruido y evitar solape con widgets flotantes
- detalle del tour más mobile-first:
  - hero más compacto
  - CTA principal/secundaria full-width en mobile
  - summary / booking / trust expuestos arriba del fold en mobile
- booking widget duplicado de forma estratégica:
  - visible arriba en mobile
  - sidebar original preservada en desktop
- quick nav horizontal preservada sin pila de botones gigantes
- padding inferior adicional para no chocar con CTA sticky

## Objetivo
Que el detalle de tour convierta mejor en vertical y que el usuario no tenga que buscar demasiado para reservar.
