# KCE Commercial Phase 87

## Objetivo
Cerrar mejor el puente comercial entre Home, Tours, Detail, Contact y Plan personalizado para que el sitio no solo se vea premium, sino que guíe mejor al viajero hacia el siguiente paso correcto.

## Cambios principales
- Home: CTA final y entry paths más claros.
- Tours: nueva sección de continuidad para elegir entre catálogo, plan personalizado y contacto.
- Plan personalizado: explicación más operativa de cómo responde KCE y cómo se conectan Bogotá, La Victoria y Cartagena.
- Tour detail: copy más consultiva para contacto y related tours.
- IA: contrato de respuesta más firme para cerrar con `Siguiente paso` cuando el usuario esté listo.

## Impacto esperado
1. Menos fricción para usuarios indecisos.
2. Mejor continuidad entre descubrimiento y cierre.
3. Mayor coherencia entre catálogo, ayuda guiada y soporte humano.
4. Base más sólida para conectar luego agentes + CRM + follow-up.

## Siguiente sprint recomendado
- Convertir `/quiz` en una ruta todavía más operacional con estados de caso.
- Añadir microcomponentes de respuesta en chat (`Resumen`, `Opciones`, `Siguiente paso`).
- Definir SOP comercial visible para Bogotá, La Victoria y Cartagena.
