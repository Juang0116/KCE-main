# P13 — Revenue Ops + Deal Timeline

## What shipped
- **/admin/revenue**: Revenue Ops dashboard (pipeline, wins, reply/paid rates, top A/B templates).
- **/api/admin/metrics/revenue-ops**: server metrics endpoint (days window).
- **Deal detail**: **/admin/deals/[id]** with combined timeline:
  - tasks
  - outbound messages (sent/replied/paid attribution)
  - events (entity_id = deal_id)
  - last conversation messages (if ticket found)

- **/api/admin/deals/[id]/timeline**: timeline endpoint used by the UI.

## Notes
- The timeline uses `events.entity_id = deal_id`. If you want richer events, ensure `logEvent(..., { entityId: dealId })` is used consistently.
- Revenue Ops template locale is inferred from outbound `metadata.locale` (if set by the template render/enqueue code).
