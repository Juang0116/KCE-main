# Fase 7.5 — Metrics to Action Loop

Objetivo: reducir el tiempo entre detectar una señal en métricas y ejecutar una acción comercial concreta.

## Qué validar
- /admin/metrics muestra el nuevo bloque `Circuito de ejecución`.
- /admin/revenue muestra accesos rápidos hacia templates, outbound y deals.
- Los enlaces rápidos conservan navegación correcta sin romper el build.

## Resultado esperado
El panel deja más claro qué hacer después de leer la señal, no solo qué está pasando.
